Tietoja sovelluksesta
jQuery TODO App on yksinkertainen tehtävienhallintasovellus, jonka avulla käyttäjä voi:
lisätä tehtäviä
merkitä tehtäviä tehdyksi
poistaa tehtäviä
nähdä tehtävät selkeästi listattuna
Sovellus käyttää jQueryä DOM-skriptaukseen ja Bootstrapia ulkoasun rakentamiseen.
